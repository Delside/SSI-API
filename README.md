## About The Project
SSI is backend part of MEAN stack project Słownik Skrótów Internetowych. Build with NodeJS and Express.js, is connecting to MongoDB database and Angular 10 frontend.

### Build With
* NodeJS
* Express.js
* MongoDB
* VS Code IDE

### Functionality
SSI API is responsible for receiving requests from Angular frontend and communication with MongoDB database.

## Getting Started
For local usage I recommend cloning the repo with command:
```sh
git clone https://github.com/Delside/SSI-API.git
```
You will need Browser and IDE with NodeJS and Express.js to run the project.

## Backend

For proper functioning, you will also need frontend part of project. It is available here: [SSI - UI](https://github.com/Delside/SSI)
